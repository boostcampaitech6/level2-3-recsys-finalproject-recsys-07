<br>
**“STEAM BUDDY” 는 사용자와 친구의 Steam ID를 이용하여 Steam 라이브러리를 분석하고 친구와 함께 즐길 수 있는 게임을 선호도 순으로 추천해주는 서비스 입니다.**

- 게임 추천 시스템: 사용자의 게임 플레이 이력을 분석하여 친구와 함께 즐길 수 있는 최적의 게임을 추천합니다
- 소셜 게이밍: 친구와 공통된 게임 취향을 발견하고 공유할 수 있는 기회를 제공
